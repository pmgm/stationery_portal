Cgiapp2_Error_Observer_Log

Cgiapp2_Error observer. Writes PHP error information to a log file.
Sample usage:  
  1.  require_once 'Cgiapp2/Error.class.php';
    2. 

  2.  require_once 'Cgiapp2/Error/Observer/Log.class.php';
    3. 

  3.  
    4. 

  4.  // Set the log file to '/tmp/errors/log'
    5. 

  5.   Cgiapp2_Error_Observer_Log::setFile('/tmp/errors/log');
    6. 

  6.  
    7. 

  7.  // Handle PHP errors
    8. 

  8.   set_error_handler(array('Cgiapp2_Error', 'handler'));
    9. 

  9.  
    10.
