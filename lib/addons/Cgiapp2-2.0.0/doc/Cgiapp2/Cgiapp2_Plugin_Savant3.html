Savant3 Template Plugin for Cgiapp2

Implements Cgiapp2_Plugin_Template_Interface to create a Savant3  plugin for Cgiapp2.
Additionally, Cgiapp2_Plugin_Savant3 implements a singleton and dynamic proxy.  This allows you to do the following:  
  1.  Cgiapp2_Plugin_Savant3::getInstance()->loadFilter('trimwhitespace');
    2. 

Registers with Cgiapp2's tmpl_path, tmpl_assign, and tmpl_fetch hooks;  registration is done with the Cgiapp2 class.

method init [line 233]

	

    

    

		bool init(
Cgiapp2
$cgiapp, mixed
$tmpl_path, [mixed
$extra_params = null])
    

    


	
		Initialize a template instance and/or set the template path

If no Cgiapp2_Plugin_Savant3 instance currently exists, it is first  initialized via getInstance() using the template path, extra  parameters, and Cgiapp2-based object instance. Savant3 is looked for in  the Cgiapp2 instance parameter 'SAVANT3_PATH'; if not found, or an error  occurs initializing Savant3, a warning is raised.
init() is used by Savant3 to set the template path.  If the template path  has not changed, nothing is done.
