Introduction

Introduction

Cgiapp2 is a PHP port of the Perl module CGI::Application. Its purpose is to serve as a framework for creating extensible, reusable web applications.

Description

Cgiapp2 is intended to make it easier to create sophisticated, reusable web-based applications. This module implements a methodology which, if followed, will make your web software easier to design, easier to document, easier to write, and easier to evolve.

Cgiapp2 builds on standard, non-proprietary technologies and techniques, such as PHP and Smarty. Like its perl ancestor, Cgiapp2 judiciously avoids employing technologies and techniques which would bind a developer to any one set of tools, operating system or web server.
